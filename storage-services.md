Storage Services \
Cloud storage services provide scalable and durable object storage for structured and unstructured data. These services are designed for backup, archival, application data storage, analytics, and content delivery.
1. AWS S3(Simple Storage Service) \
 Amazon Web Services S3 (Simple Storage Service) is an object storage service offering high durability and scalability. Data is stored in buckets and managed using objects. S3 supports multiple storage classes such as Standard, Intelligent-Tiering, Glacier, and Glacier Deep Archive for cost optimization. \
It provides features like versioning, lifecycle policies, encryption, cross-region replication, and event notifications. S3 is ideal for website hosting, data lakes, backups, media storage, and disaster recovery.
2. Azure Blob Storage \

Blob Storage is commonly used for backups, streaming content, data analytics, and storing application logs. It integrates with Azure Data Factory and Azure Synapse for analytics workloads.
3. Google cloud storage \
Google Cloud Platform Cloud Storage provides globally scalable object storage. It supports storage classes such as Standard, Nearline, Coldline, and Archive. \
It is suitable for content delivery, backups, disaster recovery, and big data analytics. The service integrates seamlessly with BigQuery and AI/ML services within GCP.
